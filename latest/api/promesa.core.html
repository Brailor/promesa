Promesa Api Documentation

Version: 0.3.0

Namespaces

  • promesa
  • core
  • protocols

promesa.core

all

(all promises)

Given an array of promises, return a promise that is fulfilled when all the items in the array are fulfilled.

any

(any promises)

Given an array of promises, return a promise that is fulfilled when first one item in the array is fulfilled.

catch

(catch p callback)(catch p type callback)

Catch all promise chain helper.

delay

(delay t)(delay t v)

Given a timeout in miliseconds and optional value, returns a promise that will fulfilled with provided value (or nil) after the time is reached.

done?

(done? p)

Returns true if promise p is already done.

error

(error p callback)

Catch operational errors promise chain helper.

finally

(finally p callback)

A chain helper that associate handler to the promise that will be called regardless if it is resolved or rejected.

fulfilled?

(fulfilled? p)

Convenience alias for the resolved? predicate.

pending?

(pending? p)

Returns true if promise p is stil pending.

promise

(promise v)

The promise constructor.

promise?

(promise? p)

Returns true if p is a primise instance.

promisify

(promisify callable)

Given a nodejs like function that accepts a callback as the last argument and return an other function that returns a promise.

reason

(reason p)

Get the rejection reason of this promise. Throws an error if the promise isn’t rejected.

rejected

(rejected v)

Return a rejected promise with provided reason.

rejected?

(rejected? p)

Returns true if promise p is already rejected.

resolved

(resolved v)

Return a resolved promise with provided value.

resolved?

(resolved? p)

Returns true if promise p is already fulfilled.

some

(some n promises)

Given an array of promises, return a promise that is fulfilled when n number of promises is fulfilled.

spread

(spread p callback)

A chain helper like then but recevies a resolved promises unrolled as parameters.

then

(then p callback)

A chain helper for promises.

timeout

(timeout p t)(timeout p t v)

Returns a cancellable promise that will be fulfilled with this promise’s fulfillment value or rejection reason. However, if this promise is not fulfilled or rejected within ms milliseconds, the returned promise is cancelled with a TimeoutError

value

(value p)

Get the fulfillment value of this promise. Throws an error if the promise isn’t fulfilled.
